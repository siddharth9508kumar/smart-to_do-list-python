# 📝 To-Do List with Task Priority (Python Project)
📌 Project Description

This is a To-Do List Manager built using Python.
The program allows users to add, remove, view, and manage tasks with priority.
It is a simple project that demonstrates basic Python concepts, functions, loops, and condition handling.

This project is useful for beginners who want to practice Python and understand how real-life applications work.

## 🚀 Features

✅ Add new task

✅ View all tasks

✅ Set task priority

✅ Delete task

✅ Mark task as completed

✅ Simple menu driven program

✅ Beginner friendly code

### 🛠️ Technologies Used

Python 3

VS Code

Terminal / Command Prompt
